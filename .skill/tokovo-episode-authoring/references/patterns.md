# Authoring Patterns

## Scene Pattern
- Organize episode by scenes with clear time windows.
- Keep each scene self-contained (devices, tracks, camera changes).

## Beat Pattern
- Use short “beats” to align dialogue, camera focus, and UI changes.

## Device Pattern
- Use one device per scene unless split-screen is required.
- Ensure device IDs match plugin registrations.

## Camera + UI Pattern
- Anchor focus should align with the active UI element in state.
