# Config and Defaults

## TokovoConfig
- Default FPS and rendering settings live in `packages/core/src/config/index.ts`.
- Use TokovoConfig as the source of truth rather than ad-hoc defaults.

## Constants
- General constants live in `packages/core/src/constants.ts`.
- Camera defaults live in `packages/core/src/types/camera.ts`.
- Audio defaults live in `packages/core/src/types/audio.ts`.
