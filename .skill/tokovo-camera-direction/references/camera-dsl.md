# Camera DSL

## Core Usage
- Use camera DSL for `set`, `animate`, `focus`, and `track` operations.
- Prefer explicit durations for animations.
- Prefer device-owned anchors for OS-level framing:
  - `device`, `app`, `keyboard`, `dynamicIsland`, `notification_banner`
