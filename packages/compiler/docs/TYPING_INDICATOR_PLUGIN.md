# TypingIndicatorPlugin

`TypingIndicatorPlugin` is a compile-time helper in `@tokovo/compiler` that generates typing-indicator behavior from authored message events.

## Purpose

Use it when you want consistent typing-indicator coverage without hand-authoring every typing span.

## Scope

The plugin is useful for repetitive conversation flows. It should not replace manual timing when an episode depends on carefully staged pauses or custom character rhythm.

## Recommended Use

- use it for standard chat pacing
- tune or disable it when a scene needs precise dramatic timing
- verify the generated feel in preview and render
