# @tokovo/episodes

`@tokovo/episodes` is the canonical content surface for Tokovo.

It exports:

- episode definitions
- production and test episode catalogs
- validation helpers

## Flow

1. author the cinematic scene directly in `*.episode.ts`
2. register it in a catalog
3. preview via `video-runner`
4. render locally through `video-runner` or `@tokovo/render-service`
