## Live2D Demo Assets

This directory contains official Live2D assets vendored for Tokovo's internal
reactor demo.

### Source

- Cubism SDK for Web package: `CubismSdkForWeb-5-r.5.zip`
- Download page: `https://www.live2d.com/en/sdk/download/web/`
- Core runtime URL:
  `https://cubism.live2d.com/sdk-web/cubismcore/live2dcubismcore.min.js`
- Retrieved: `2026-04-18`

### Included assets

- `haru/`: official `Haru` sample model extracted from the SDK package
- `../scripts/live2dcubismcore.min.js`: official Cubism Core for Web runtime
- `UPSTREAM-LICENSE.md`: upstream SDK license summary from the downloaded package

### License / usage notes

- `Haru` is listed in the SDK package under the Live2D Free Material License.
- The SDK package states that business users above the revenue threshold must
  obtain a Cubism SDK Release License before publishing content.
- Sample-model terms still apply to `Haru` separately from the SDK licenses.

Use these assets for internal preview and evaluation unless legal review has
cleared the final release path.
