# 🎬 Tokovo Cinematography Style Guide

<Callout type="warning">
<strong>v1 PRESETS ARE SACRED. Do not modify their semantics.</strong>
</Callout>

## v1 → v2 Evolution Plan

### v1 Goal

> **Make 90% of chat-based stories feel cinematic, readable, and emotionally intentional.**

### v1 Characteristics

- Deterministic
- Rule-based
- Small preset vocabulary
- Mostly reactive camera
- One active anchor at a time
- No AI "creativity", only correctness

### v1 Camera Personality

> A careful human operator who reacts, but does not anticipate.

**v1 does NOT predict emotion. It responds to events.**

---

### v2 Goal

> **Make the camera feel anticipatory, intelligent, and narratively aware.**

v2 does **not** replace v1. It *layers on top* of it.

### v2 Unlocks

- Pre-emptive camera movement
- Multi-anchor awareness
- Emotional state continuity
- Style profiles (genre-level)
- AI-assisted intent selection
- Micro-variation to avoid repetition

---

## Evolution Axes

| Axis      | v1       | v2                    |
| --------- | -------- | --------------------- |
| Anchor    | Single   | Primary + secondary   |
| Timing    | Reactive | Anticipatory          |
| Presets   | Static   | Parameterized         |
| Emotion   | Implicit | Explicit              |
| Style     | Global   | Profile-driven        |
| Variation | None     | Controlled randomness |

## v2 System Requirements

Do **not** enable v2 presets until these exist:

### 1️⃣ Emotional State Model

```typescript
emotion = {
  tension: 0..1,
  intimacy: 0..1,
  volatility: 0..1
}
```

### 2️⃣ Multi-Anchor Awareness

Camera can reason about primary + secondary anchors.

### 3️⃣ Preset Parameterization

```typescript
impact(emotion) → { scale, shake }
```

### 4️⃣ Controlled Randomness

Tiny variations (±3%) to avoid repetition. Seeded. Deterministic.
