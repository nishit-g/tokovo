# Plugin System

Tokovo uses a **plugin architecture** for apps. Each app (WhatsApp, Instagram, etc.) is a self-contained plugin.

## Why Plugins?

| Benefit | How |
|---------|-----|
| **Self-contained** | Each app owns its UI, state, sounds |
| **Explicit registration** | Plugins register with `createPluginRegistry()` |
| **Extensible** | Add new apps without modifying core |
| **Typed** | Full TypeScript support |

## Canonical Plugin System (Recommended)

The canonical plugin system uses explicit factory-based registration for testability and composition control.

### AppPlugin Interface

```typescript
import type { AppPlugin, PluginSchema } from "@tokovo/core";

interface AppPlugin {
  // Identity
  id: string;              // "app_whatsapp"
  name: string;            // "WhatsApp"
  version: string;         // "2.0.0"

  // Capabilities
  capabilities: AppCapability[];  // ["messaging", "typing", ...]
  schema: PluginSchema;           // Content kinds, event types, limits

  // Components
  view: AppViewComponent;   // React component
  reducer: AppReducer;      // State reducer

  // Branding
  icon?: string;           // Icon path
  primaryColor?: string;   // "#25D366"

  // Audio
  sounds?: Record<string, string>;  // Sound mappings
  notificationSound?: string;       // Default notification sound
  
  // State
  defaultState?: unknown;  // Initial app state
}
```

### Creating a Plugin

```typescript
// packages/apps-whatsapp/src/plugin.ts
import { canonical } from "@tokovo/core";
import type { AppPlugin, PluginSchema } from "@tokovo/core";
import { whatsappReducer } from "./runtime";
import { WhatsAppView } from "./ui";

const { defineAppPlugin } = canonical;

const schema: PluginSchema = {
  contentKinds: ["text", "image", "video", "voice", "sticker"],
  eventTypes: ["MESSAGE", "TYPING", "READ", "REACTION", "NAVIGATE"],
  limits: {
    maxTextLength: 65536,
    maxCaptionLength: 1024,
  },
};

export const WhatsAppPlugin: AppPlugin = defineAppPlugin({
  id: "app_whatsapp",
  name: "WhatsApp",
  version: "2.0.0",
  
  capabilities: ["messaging", "typing", "read_receipts", "reactions"],
  schema,
  
  view: WhatsAppView,
  reducer: whatsappReducer,
  
  icon: "whatsapp-icon.png",
  primaryColor: "#25D366",
  
  sounds: {
    message_in: "whatsapp-received.mp3",
    message_out: "whatsapp-sent.mp3",
  },
  
  notificationSound: "whatsapp-notification.mp3",
});
```

### Using Plugins with the Engine

```typescript
// Explicit plugin registration
import { createPluginRegistry, createEngine } from "@tokovo/core";
import { WhatsAppPlugin } from "@tokovo/apps-whatsapp";
import { TwitterPlugin } from "@tokovo/apps-twitter";

// Create registry and register plugins
const plugins = createPluginRegistry();
plugins.register(WhatsAppPlugin);
plugins.register(TwitterPlugin);

// Create engine with plugins
const engine = createEngine({
  plugins,
  fps: 30,
  validation: "strict",
});

// Build world state
const world = engine.buildWorld(initialWorld, events, frame);
```

### Video-Runner Helper

For convenience, video-runner provides a pre-configured registry:

```typescript
// apps/video-runner/src/engine.ts
import { createTokovoPluginRegistry } from "./engine";

const plugins = createTokovoPluginRegistry();
// → Includes WhatsApp, Twitter, Instagram, Phone
```

## Adding a New App

1. **Create package** — `packages/apps-newapp/`
2. **Define plugin** — Export `NewAppPlugin: AppPlugin`
3. **Create reducer** — Handle your event types  
4. **Create view** — React component
5. **Add to video-runner** — Register in `engine.ts`

```typescript
// packages/apps-newapp/src/plugin.ts
import { canonical } from "@tokovo/core";
import type { AppPlugin } from "@tokovo/core";

export const NewAppPlugin: AppPlugin = canonical.defineAppPlugin({
  id: "app_newapp",
  name: "New App",
  version: "1.0.0",
  capabilities: ["messaging"],
  schema: { contentKinds: ["text"], eventTypes: ["MESSAGE"] },
  view: NewAppView,
  reducer: newAppReducer,
});
```

```typescript
// apps/video-runner/src/engine.ts
import { NewAppPlugin } from "@tokovo/apps-newapp";

plugins.register(NewAppPlugin);
```

## PluginRegistry API

```typescript
import { createPluginRegistry } from "@tokovo/core";

const plugins = createPluginRegistry();

// Register a plugin
plugins.register(MyPlugin);

// Get a plugin
const plugin = plugins.get("app_whatsapp");

// Check capabilities
plugins.hasCapability("app_whatsapp", "messaging"); // true
plugins.supportsContentKind("app_whatsapp", "voice"); // true
plugins.supportsEventType("app_whatsapp", "MESSAGE"); // true

// Get all plugins
const allPlugins = plugins.all();
const allIds = plugins.ids();

// Get reducers (for engine)
const reducers = plugins.getReducers();
```

## Legacy Plugin System (Deprecated)

> ⚠️ **Deprecated**: The following API is maintained for backward compatibility only.
> Use the canonical `defineAppPlugin()` and `createPluginRegistry()` instead.

### Legacy TokovoPlugin Interface

```typescript
interface TokovoPlugin {
  id: string;
  name: string;
  version: string;
  appView?: AppViewComponent;
  reducer?: AppReducer;
  eventTypes?: string[];
  sounds?: Record<string, string>;
}
```

### Legacy Registration

```typescript
import { PluginManager, definePlugin } from "@tokovo/core";

const MyPlugin = definePlugin({
  id: "app_myapp",
  name: "My App",
  reducer: myReducer,
});

PluginManager.register(MyPlugin);
```

### Legacy ReducerRegistry

```typescript
import { ReducerRegistry } from "@tokovo/core";

// Still works but deprecated
ReducerRegistry.registerAppReducer("app_myapp", myReducer);
```

### Legacy replay()

```typescript
import { replay } from "@tokovo/core";

// Still works but deprecated
const world = replay(initialWorld, events, frame);
```

## Migration Guide

### From Legacy to Canonical

1. **Replace `definePlugin`** with `defineAppPlugin`
2. **Replace `PluginManager.register`** with `createPluginRegistry().register`
3. **Replace `replay()`** with `createEngine().buildWorld()`
4. **Add `capabilities` and `schema`** to plugin definition

```diff
- import { definePlugin, PluginManager } from "@tokovo/core";
+ import { canonical } from "@tokovo/core";
+ import type { AppPlugin } from "@tokovo/core";

- export const MyPlugin = definePlugin({
-   id: "app_myapp",
-   eventTypes: ["MESSAGE_RECEIVED"],
- });
+ export const MyPlugin: AppPlugin = canonical.defineAppPlugin({
+   id: "app_myapp",
+   capabilities: ["messaging"],
+   schema: { contentKinds: ["text"], eventTypes: ["MESSAGE"] },
+ });

- PluginManager.register(MyPlugin);
+ // Register in video-runner engine.ts
```
