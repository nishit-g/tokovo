## DSL Usage

### Incoming Call

```typescript
d.beat("incoming-call", b => {
    // Use CALL events directly
    // Incoming call from Bob
});
```

### Timeline Events

```typescript
// Incoming call
{
  at: 100,
  kind: "CALL",
  type: "INCOMING",
  deviceId: "phone",
  callerId: "bob",
  callerName: "Bob Smith",
  callerAvatar: "/avatars/bob.png",
  isVideo: false,
  callType: "voice",
  displayMode: "fullscreen"
}

// Answer
{ at: 150, kind: "CALL", type: "ANSWER", deviceId: "phone" }

// Decline
{ at: 200, kind: "CALL", type: "DECLINE", deviceId: "phone" }

// End call
{ at: 300, kind: "CALL", type: "END", deviceId: "phone" }

// Toggle controls
{ at: 180, kind: "CALL", type: "TOGGLE_MUTE", deviceId: "phone" }
{ at: 200, kind: "CALL", type: "TOGGLE_SPEAKER", deviceId: "phone" }
{ at: 220, kind: "CALL", type: "TOGGLE_HOLD", deviceId: "phone" }
```

### Semantic Anchors

```typescript
type PhoneAnchorId =
  | "callPoster"      // Contact poster image
  | "acceptButton"    // Green answer button
  | "declineButton"   // Red decline button
  | "callTimer"       // Active call timer
  | "muteButton"      // Mute control
  | "device";         // Full frame fallback
```

## Call States

| State | Description | UI |
|-------|-------------|-----|
| `incoming` | Call is ringing | Full-screen with accept/decline |
| `ringing` | Outgoing call ringing | Timer shows "Calling..." |
| `active` | Call connected | Timer counting, control grid |
| `declined` | Call was declined | Brief flash, then dismiss |
| `ended` | Call ended | Brief flash, then dismiss |
