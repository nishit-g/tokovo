# Apps

## Using Apps

```typescript
// Import to auto-register the app
import "@tokovo/apps-whatsapp";

// Then use in your events
const events = [
  dsl.messages.receive(0, "conv_1", "Alice", "Hello!"),
  dsl.messages.send(60, "conv_1", "Hi there!"),
];
```

Each import:
- Registers the app's **reducer** with `ReducerRegistry`
- Registers the app's **view** with `PluginManager`
- Registers the app's **sounds**, **behaviors**, and **anchors**

---

## App Architecture

Every app consists of:

| Component | Purpose |
|-----------|---------|
| `plugin.ts` | Registration with PluginManager |
| `runtime.ts` | Reducer for state updates |
| `ui.tsx` | React components for rendering |
| `behaviors.ts` | Camera intent mappings |
| `anchors.ts` | Semantic anchor providers |
| `notification-adapter.ts` | Notification formatting |
