# WorldState

WorldState is the **complete snapshot** of everything at a point in time.

## The Golden Rule

> **Frame = f(initialWorld, events, t)**

At any frame t, the world state is computed by:
1. Start with `initialWorld`
2. Apply all events where `event.at <= t`
3. Result is the WorldState at frame t

This is **deterministic** — same inputs always produce same output.

## WorldState Structure

```typescript
interface WorldState {
  // All devices
  devices: Record<DeviceId, DeviceState>;
  
  // All conversations (shared across devices)
  conversations: Record<ConversationId, ConversationState>;
  
  // App-specific state
  appState: Record<AppId, any>;
  
  // Camera state
  camera: CameraState;
  
  // Audio state
  audio: AudioState;
  
  // Internal cursor
  _eventCursor: number;
}
```

## DeviceState

The complete state of a single device:

```typescript
interface DeviceState {
  id: string;                           // "alice_phone"
  profileId: string;                    // "iphone16" | "pixel"
  ownerName?: string;                   // "Alice"
  isLocked: boolean;                    // Show lockscreen?
  foregroundAppId?: string;             // "app_whatsapp"
  
  // Notifications
  notifications: Notification[];         // All notifications
  notificationCenter?: NotificationCenterState;
  dynamicIsland?: DynamicIslandState;   // iOS Dynamic Island state
  statusBarIcons?: StatusBarIcon[];     // Android status bar
  
  // Background activity
  backgroundApps?: BackgroundAppState[];
  call?: CallState;                     // Incoming/active call
  
  // UI configuration
  homeScreen?: HomeScreenConfig;
  theme?: DeviceTheme;
  sound?: { activeSoundId?: string };
}
```

## CameraState

```typescript
interface CameraState {
  baseView: "APP_VIEW" | "LOCKSCREEN" | "HOMESCREEN";
  activeDeviceId: string;
  
  // Multi-device layout
  layout: {
    mode: "SINGLE" | "SPLIT" | "STACK";
    primaryDeviceId: string;
    secondaryDeviceId?: string;
  };
  
  // Active effects (ZOOM, PAN, SHAKE, FOCUS, RESET)
  activeEffects: ActiveCameraEffect[];
  
  // Computed transform
  transform: CameraTransform;
  
  // Per-device transforms
  deviceTransforms: Record<string, CameraTransform>;
}

interface CameraTransform {
  scale: number;        // 1.0 = no zoom
  translateX: number;
  translateY: number;
  rotation: number;
  originX: number;      // 0-1
  originY: number;
  shakeX: number;
  shakeY: number;
}
```

## Building World at Frame

```typescript
import { replay } from "@tokovo/core";

// Compute world state at frame 100
const world = replay(initialWorld, events, 100);
```

## Accessing State

```typescript
// Get device
const device = world.devices["phone"];

// Get foreground app
const appId = device.foregroundAppId;

// Get notification center
const notifCenter = device.notificationCenter;
const headsUp = notifCenter?.headsUp;

// Get conversation
const conv = world.conversations["dm_bob"];

// Get camera transform
const { scale, translateX, translateY } = world.camera.transform;

// Get active sounds
const sounds = Object.values(world.audio.activeSounds);
```

## Determinism Notes

- Is this pure? **Yes** — Computed from events
- Is ordering stable? **Yes** — Events are sorted by `at` field
- Is randomness seeded? **Yes** — Camera shake uses seeds
- Is scrubbing supported? **Yes** — Any frame, any time
