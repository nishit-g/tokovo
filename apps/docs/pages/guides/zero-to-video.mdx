## The Steps

<Steps>

### Clone and Install

```bash
# Clone the repository
git clone https://github.com/your-org/tokovo.git
cd tokovo

# Install all dependencies
pnpm install

# Build all packages
pnpm build
```

This takes ~2 minutes on first run.

### Start the Video Runner

```bash
# Start Remotion Studio (development server)
pnpm --filter video-runner dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

You should see the Remotion Studio with an example video composition.

### Create Your First Episode

Create a new file `apps/video-runner/src/MyFirstVideo.tsx`:

```tsx
import React from "react";
import { AbsoluteFill } from "remotion";
import { TokovoRenderer, replay } from "@tokovo/renderer";
import { useCurrentFrame } from "remotion";
import { dsl } from "@tokovo/dsl";

// Import WhatsApp plugin
import "@tokovo/apps-whatsapp";

// Define your story as events
const events = [
  // Set up the conversation
  dsl.os.setTime(0, Date.now()),
  dsl.os.setBattery(0, 87),
  
  // First message received
  dsl.messages.receive(30, "conv_1", "Alex", "Hey! Are you free tonight?"),
  dsl.camera.anchorFocus(30, "lastMessage", "message"),
  
  // Camera holds for reading
  dsl.camera.hold(60, 20),
  
  // User replies
  dsl.messages.send(120, "conv_1", "Yeah, what's up?"),
  dsl.camera.anchorFocus(120, "lastMessage", "snap"),
  
  // Final hold
  dsl.camera.hold(150, 30),
];

// Initial world state
const initialWorld = {
  devices: {
    phone: {
      id: "phone",
      profileId: "iphone16",
      isLocked: false,
      foregroundAppId: "app_whatsapp",
      os: { clock: Date.now(), battery: 87 },
    },
  },
  conversations: {
    conv_1: {
      id: "conv_1",
      name: "Alex",
      messages: [],
    },
  },
  appState: {},
  camera: { baseView: "APP_VIEW", activeDeviceId: "phone" },
  audio: { activeSounds: {}, buses: {} },
};

export const MyFirstVideo: React.FC = () => {
  const frame = useCurrentFrame();
  
  // Compute world state at current frame
  const world = replay(initialWorld, events, frame);
  
  return (
    <AbsoluteFill style={{ backgroundColor: "#000" }}>
      <TokovoRenderer
        world={world}
        t={frame}
        focusDeviceId="phone"
        directorEnabled={true}
      />
    </AbsoluteFill>
  );
};

// Duration in frames (30fps = 6 seconds)
export const myFirstVideoDuration = 180;
```

### Register the Composition

Open `apps/video-runner/src/Root.tsx` and add your composition:

```tsx
import { MyFirstVideo, myFirstVideoDuration } from "./MyFirstVideo";

// Add to the list of compositions
<Composition
  id="MyFirstVideo"
  component={MyFirstVideo}
  durationInFrames={myFirstVideoDuration}
  fps={30}
  width={1080}
  height={1920}
/>
```

### View Your Video

1. Refresh the browser at [http://localhost:3000](http://localhost:3000)
2. Select "MyFirstVideo" from the composition list
3. Press **Space** to play!

### Export to MP4 (Optional)

```bash
# Render to MP4
pnpm --filter video-runner render MyFirstVideo

# Output: out/MyFirstVideo.mp4
```

</Steps>

## Understanding What Happened

```
Your Code (events[])
       │
       ▼
replay(initialWorld, events, frame)
       │
       ▼
WorldState at frame N
       │
       ▼
TokovoRenderer → Pixel-Perfect Video
```

| Step | What Happens |
|------|--------------|
| **Events** | Define WHEN things happen (frame numbers) |
| **replay()** | Computes the complete state at any frame |
| **TokovoRenderer** | Renders the state to React/Remotion |
| **Remotion** | Exports to MP4/WebM |

## Troubleshooting

| Issue | Solution |
|-------|----------|
| "Module not found" | Run `pnpm build` first |
| Black screen | Check `foregroundAppId` is set correctly |
| No camera movement | Ensure `directorEnabled={true}` |
| Composition not listed | Check you added it to `Root.tsx` |
