# Data Flow

```text
packages/episodes scene code
  -> ir (devices + appSnapshots + initialViews + events)
  -> compiler (schema validate + migrate + hydrate)
  -> renderer
  -> video-runner
```

Primary artifacts:

- `packages/episodes/src/production/index.ts`
- `packages/episodes/src/showcases/index.ts`
- `packages/episodes/src/v2/index.ts`

Frame-0 data flow is strict:

- devices describe only device/OS bootstrap
- app snapshots are plugin-owned historical state
- initial views are plugin-owned routes at frame 0
- runtime events start after bootstrap and mutate live state
