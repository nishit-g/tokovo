# Tokovo

> **Programmable phone simulation for cinematic storytelling**

Tokovo transforms structured episode definitions into high-fidelity phone UI videos using React and Remotion. Instead of manually animating fake chats in editing tools, describe your story as **events over time**, and Tokovo renders the entire episode deterministically.

Built for **content creators**, **AI-driven storytelling**, and **modular mobile OS simulations** — from scripted narratives to brand collaborations and experimental formats.

## Key Features

**Deterministic Simulation**  
Every frame is reproducible from episode data. No runtime surprises.

**Event-Driven Storytelling**  
Messages, typing indicators, notifications, app transitions—all timeline events.

**Modular Plugin Architecture**  
Add new devices (iPhone, Pixel), apps (WhatsApp, Instagram), or effects without touching core.

**React + Remotion Rendering**  
Leverage the full power of React components with Remotion's video engine.

**AI-Friendly Episode Format**  
Structured JSON/TypeScript episodes make it easy for LLMs to generate storylines.

## Quick Example

Here's how simple it is to create a phone conversation:

```typescript
import { episode } from "@tokovo/dsl";
import { WhatsAppTrackBuilder } from "@tokovo/apps-whatsapp";

episode("my-story", { fps: 30, duration: "10s" })
  .device("phone", "iphone16", {
    app: "app_whatsapp",
    conversations: [{ id: "dm_alice", name: "Alice" }],
  })
  .track(
    "app_whatsapp",
    (getOrder) => new WhatsAppTrackBuilder(30, "phone", "dm_alice", getOrder),
    (wa) => {
      wa.at("1s").receive("Alice", "Hey! How are you?");
      wa.at("3s").typing("Alice");
      wa.at("5s").receive("Alice", "Want to grab coffee?");
    },
  )
  .camera((cam) => {
    cam.at("0s").focus("app_whatsapp:message", { scale: 1.05, duration: "0.4s" });
    cam.at("4s").animate({ scale: 1.2, duration: "0.5s" });
  });
```

This episode produces a fully-rendered video with typing animations, message bubbles, and camera movements—all from declarative code.

## What Makes Tokovo Different

- **Data-driven**: Episodes are data, not code. Perfect for batch generation and AI workflows.
- **Layered architecture**: Core engine stays pure and minimal. Complexity lives in devices/apps.
- **Composable**: New devices or apps never require core changes.
- **Cinematic focus**: Optimized for video rendering, not interactive apps.
