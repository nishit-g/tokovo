# Showcase

The showcase catalog is the fastest way to understand Tokovo's v1 phone-simulation surface. It contains full app demos, system realism tests, theme passes, and creator-story baselines that can be previewed or rendered locally.

## Render One

```bash
EPISODE_ID=v2-creator-series-showcase pnpm --filter video-runner render:fast
```

Outputs:

```text
out/v2-creator-series-showcase.mp4
out/v2-creator-series-showcase.manifest.json
```

## Catalog Profiles

Use `release` for creator-facing episodes and `studio` for the full app/system showcase surface.

```bash
TOKOVO_EPISODE_CATALOG_PROFILE=studio pnpm --filter video-runner dev
```

## Quality Checklist

Before calling a showcase v1-ready:

- `pnpm validate` passes.
- The episode renders through `render:fast`.
- All referenced assets exist under `apps/video-runner/public`.
- Camera targets resolve to semantic anchors instead of app fallback boxes.
- The final MP4 and manifest are kept out of git unless they are intentional docs assets.
