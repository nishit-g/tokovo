# Installation

Get the Tokovo monorepo running locally.

## Prerequisites

- Node.js 18+
- `pnpm`
- Git

## Clone

```bash
git clone <YOUR_REPO_URL>
cd tokovo
```

## Install Dependencies

```bash
pnpm install
```

## Main Apps

- `apps/web`: marketing site
- `apps/video-runner`: Remotion render app
- `apps/docs`: documentation site

## Main Package Groups

- compiler and authoring: `ir`, `core`, `dsl`, `compiler`, `episodes`
- device and OS: `devices`, `device-camera`, `device-keyboard`, `device-notifications`, `overlay`
- app plugins: `apps-whatsapp`, `apps-imessage`, `apps-x`, `apps-linkedin`, `apps-snapchat`, `apps-teams`, `apps-typewriter`
- rendering and media: `react`, `renderer`, `background`, `voice`

## Useful Commands

```bash
pnpm build
pnpm test
pnpm build:video
pnpm --filter docs build
```

## Local Entry Points

```bash
pnpm --filter video-runner dev
pnpm --filter docs dev
pnpm --filter web dev
```
