# First Episode

The first Tokovo episode should be authored directly in code.

```ts
import { episode } from "@tokovo/dsl";

const ep = episode("my-episode", { fps: 30, duration: "8s", title: "My Episode" })
  .device("phone", "iphone16", { app: "app_whatsapp" })
  .snapshot("app_whatsapp", "phone", {
    conversations: [{ id: "dm_alex", name: "Alex", messages: [] }],
  })
  .view("app_whatsapp", "phone", {
    screen: "chat",
    conversationId: "dm_alex",
  });
```

After that, keep timing, camera, and scene sequencing in code.

Use `snapshot()` and `view()` for all frame-0 app bootstrap. Old device-owned history patterns are not supported.
