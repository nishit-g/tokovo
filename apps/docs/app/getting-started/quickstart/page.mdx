# Quickstart

This path verifies the repo, previews the renderer, and exports one current v1 showcase episode.

## 1. Install

```bash
pnpm install
```

## 2. Validate the Episode Catalog

```bash
pnpm validate
pnpm --filter @tokovo/episodes test
```

`pnpm validate` boots the studio runtime and validates every registered episode definition.

## 3. Open the Renderer

```bash
pnpm --filter video-runner dev
```

Use the Remotion Studio UI to select an episode composition and scrub the timeline.

## 4. Render a Local MP4

```bash
EPISODE_ID=v2-creator-series-showcase pnpm --filter video-runner render:fast
```

The fast renderer writes:

```text
out/v2-creator-series-showcase.mp4
out/v2-creator-series-showcase.manifest.json
```

Try a shorter app-specific showcase:

```bash
EPISODE_ID=whatsapp-flagship-v2 pnpm --filter video-runner render:fast
EPISODE_ID=x-flagship-v2 pnpm --filter video-runner render:fast
EPISODE_ID=typewriter-flagship-v2 pnpm --filter video-runner render:fast
```

## 5. Know the Main Catalogs

| Catalog | Includes |
| --- | --- |
| `release` | public story and v2 creator episodes |
| `studio` | release catalog plus app, system, and test showcases |

Set the catalog profile when needed:

```bash
TOKOVO_EPISODE_CATALOG_PROFILE=studio pnpm --filter video-runner dev
```
