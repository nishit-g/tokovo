# Performance

Tokovo performance is mostly about keeping runtime replay deterministic and keeping render-time work explicit.

## Main Levers

- keep episode catalogs explicit
- avoid hidden import-time side effects
- use preview and fast-render smoke episodes to catch regressions early

## Practical Checks

- `pnpm build:video`
- `pnpm --filter docs build`
- `pnpm --filter video-runner render:fast`
