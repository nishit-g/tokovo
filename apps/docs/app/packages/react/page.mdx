# @tokovo/react

Shared React runtime bindings and registry helpers for Tokovo.

## Responsibilities

- React-facing runtime helpers
- provider and registry integration
- app-surface wiring used by render apps

This package should connect runtime state to React without absorbing app-specific logic that belongs in plugin packages.

## Key Exports

- provider and hooks: `TokovoProvider`, `useWorld`, `useDevice`, `useAppState`, `useLayout`
- keyboard-aware helpers: `KeyboardAwareView`, `ScrollableContent`, `useKeyboardState`
- app surface wrapper: `AppSurface`
- runtime registry creation: `createTokovoRegistries`
