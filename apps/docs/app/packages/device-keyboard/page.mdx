# @tokovo/device-keyboard

Keyboard runtime support and keyboard-aware UI helpers.

## Responsibilities

- keyboard runtime behavior
- keyboard registration helpers
- keyboard UI layouts and components
- keyboard authoring support

## Key Exports

- runtime state and selectors
- `KeyboardTrackBuilder`
- keyboard components: `Keyboard`, `Key`, `KeyRow`
- layout presets: `qwertyLayout`, `numericLayout`, `phoneLayout`, `emailLayout`
- plugin registration: `registerKeyboardPlugin`
