# @tokovo/core

The deterministic runtime layer.

## Responsibilities

- world-state computation
- replay and runtime state transitions
- engine registries
- shared runtime types
- audio primitives

## Boundary

`@tokovo/core` must stay headless and should not depend on React or renderer code.

## Key Exports

- replay and preparation: `replayIncremental`, `prepareEpisode`, `runEpisode`
- engine helpers: `createEventIndex`, `createKeyframedEventIndex`, `createStateCache`
- registries: `createEngine`, `createEngineRegistries`, `createReducerRegistry`
- anchors: `AnchorRegistryClass`, `createAnchorRegistry`
- config: `TokovoConfig`, `EngineConfig`
- validation and logging helpers
- shared runtime and event types
