# @tokovo/devices

The device profile and device-shell layer for Tokovo.

## Responsibilities

- device profiles
- device registry wiring
- device shell support used by renderer and runtime apps

## Key Exports

- registries: `createDeviceRegistries`, `DeviceRegistryClass`
- DSL: `DeviceTrackBuilder`
- reducer: `deviceReducer`
- plugin registration: `registerDevicesPlugin`
- built-in profiles: `iPhone16Profile`, `PixelProfile`
