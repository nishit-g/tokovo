# @tokovo/episodes

> **Example Episodes & Learning Resources**

Priority: **P2** (Examples, Templates, Learning)

`@tokovo/episodes` is Tokovo's collection of production-ready example episodes, feature showcases, and episode templates. Use these to learn Tokovo's DSL, understand best practices, and jumpstart your own video projects.

---

## Installation

```bash
npm install @tokovo/episodes
# or
pnpm add @tokovo/episodes
# or
yarn add @tokovo/episodes
```

## Using Episodes in Your Code

### Import and Use

```typescript
import { trackDemoV2, bakchodiBros, cameraShowcase } from '@tokovo/episodes/production'
import { runEpisode, TokovoConfig } from "@tokovo/core";
import { createTokovoRegistries, PluginManagerClass } from "@tokovo/react";
import { createDeviceRegistries, registerDevicesPlugin } from "@tokovo/devices";

// Get state at specific time
const registries = createTokovoRegistries()
const deviceRegistries = createDeviceRegistries()
registerDevicesPlugin(registries, deviceRegistries)

const rendererRegistries = {
  plugins: registries.plugins,
  devices: deviceRegistries,
}
const pluginManager = new PluginManagerClass(registries.plugins)

const worldAt15s = runEpisode(bakchodiBros, 450, {
  mode: "preview",
  registries: registries.engine,
  config: TokovoConfig,
})

// Render in Remotion
import { TokovoRenderer } from '@tokovo/renderer'

function MyComposition() {
  const frame = useCurrentFrame()
  const worldState = runEpisode(bakchodiBros, frame, {
    mode: "preview",
    registries: registries.engine,
    config: TokovoConfig,
  })

  return (
    <TokovoRenderer
      world={worldState}
      t={frame}
      pluginManager={pluginManager}
      registries={rendererRegistries}
    />
  )
}
```

### Access Episode Metadata

```typescript
import productionEpisodes from "@tokovo/episodes/production";
import { createEpisodeRegistry } from "@tokovo/episodes";

const episodeRegistry = createEpisodeRegistry();
productionEpisodes.forEach((ep) => episodeRegistry.register(ep));

// Get specific episode
const episode = episodeRegistry.get("bakchodi-bros");

console.log(episode.meta.title); // "Bakchodi Bros 🤪"
console.log(episode.config.durationInFrames); // 2700
console.log(episode.config.format); // "1080x1920"
```

## Episode Patterns

Common patterns found across production episodes:

### 1. **Opening Shot Pattern**

```typescript
.camera(cam => {
  cam.at("0s").set({ scale: 1 });
  cam.at("1s").animate({
    scale: 1.08,
    duration: "0.5s",
    easing: "easeOut"
  });
})
```

Wide view → subtle zoom to draw attention.

### 2. **Message Reveal Pattern**

```typescript
// Typing indicator
wa.span("7s", "10s").typing("them");

// Dramatic message
wa.at("10s").receive("Sarah", "I have something important to tell you...");

// Camera focus on reveal
cam.at("10s").focus("app_whatsapp:message", { scale: 1.15, duration: "0.4s" });
```

Build suspense with typing, then focus camera on reveal.

### 3. **Shock/Impact Pattern**

```typescript
// Big news arrives
wa.at("20s").receive("Sarah", "I GOT THE JOB! 🎉");

// Camera punch zoom + shake
cam.at("20s").animate({
  scale: 1.3,
  y: -40,
  duration: "0.8s",
  easing: "cinematic",
});
cam.at("20.5s").shake({
  intensityX: 5,
  intensityY: 4,
  frequency: 20,
  decay: 0.85,
  duration: "0.4s",
});
```

Zoom + shake for dramatic emphasis.

### 4. **Conversation Pacing Pattern**

```typescript
// Message received
wa.at("1s").receive("Alex", "Hey!");

// User thinks (2 second pause)

// User types
wa.span("3s", "5s").typing("me");

// User sends
wa.at("5s").send("Hi there!");
```

Natural conversation rhythm: receive → pause → typing → send.

### 5. **Closing Pattern**

```typescript
// Final exchange
wa.at("36s").receive("Sarah", "Can't wait! 💋❤️");

// Read receipt
wa.at("38s").read();

// Camera reset
cam.at("40s").reset({
  duration: "2s",
  easing: "easeOut",
});
```

Gentle resolution with smooth camera return to baseline.

## Episode Structure Template

All production episodes follow this structure:

```typescript
import { defineEpisode } from "@tokovo/episodes";
import { episode } from "@tokovo/dsl";
import { WhatsAppTrackBuilder } from "@tokovo/apps-whatsapp";

export default defineEpisode({
  // 1. Metadata
  meta: {
    id: "episode-id",
    title: "Display Title",
    description: "What this episode shows",
    category: "production",
    tags: ["whatsapp", "drama"],
  },

  // 2. Video config
  config: {
    format: "1080x1920",
    durationInFrames: 900, // 30s * 30fps
    apps: ["app_whatsapp"],
  },

  // 3. Content builder
  build: () =>
    episode("episode-id", {
      fps: 30,
      duration: "30s",
    })
      // Device setup
      .device("phone", "iphone16", {
        app: "app_whatsapp",
        conversations: [
          { id: "dm_contact", name: "Contact", avatar: "/avatar.jpg" },
        ],
        os: {
          time: new Date("2024-12-17T19:30:00"),
          battery: 87,
          network: "5G",
        },
      })

      // WhatsApp messages
      .track(
        "app_whatsapp",
        (getOrder) => new WhatsAppTrackBuilder(30, "phone", "dm_contact", getOrder),
        (wa) => {
          wa.at("1s").receive("Contact", "Message here");
          wa.span("3s", "5s").typing("me");
          wa.at("5s").send("Reply here");
        },
      )

      // Camera movements
      .camera((cam) => {
        cam.at("0s").set({ scale: 1 });
        cam.at("1s").animate({ scale: 1.1, duration: "0.5s" });
      })

      // Audio
      .audio((audio) => {
        audio.span("0s", "30s").bgm("track", { volume: 0.12 });
      })

      // Markers for navigation
      .mark("intro", "1s")
      .mark("reveal", "15s")
      .section("opening", "0s", "15s")
      .section("conclusion", "15s", "30s")

      .build(),
});
```
