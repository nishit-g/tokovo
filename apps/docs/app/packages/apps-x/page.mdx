# @tokovo/apps-x

The X plugin package for Tokovo.

## Responsibilities

- X runtime reducer and initial state
- X snapshot and initial-view schema ownership
- snapshot/view validation and migration hooks
- X React views
- feed and thread layout support
- anchor integration
- authoring helpers and track builder support

## Runtime Registration

`video-runner` registers this plugin explicitly at startup. The package should remain a normal plugin package rather than depending on import-time side effects.

## Key Exports

- plugin registration: `registerXPlugin`
- runtime state and selectors
- `XTrackBuilder`
- type exports for users, tweets, notifications, DM threads, and routes
