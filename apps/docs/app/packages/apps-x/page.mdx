# @tokovo/apps-x

The X (Twitter) app simulation plugin. Provides UI views, reducer logic, DSL track builder, and optional lowering for compiled episodes.

## Install

```bash
pnpm add @tokovo/apps-x
```

## Register

```typescript
import { createTokovoRegistries, PluginManagerClass } from "@tokovo/react";
import { registerXPlugin } from "@tokovo/apps-x";

const registries = createTokovoRegistries();
const pluginManager = new PluginManagerClass(registries.plugins);

registerXPlugin(pluginManager);
```

## Use In Episodes

```typescript
import { episode } from "@tokovo/dsl";
import { XTrackBuilder } from "@tokovo/apps-x";

episode("x-demo", { fps: 30, duration: "20s" })
  .device("phone", "iphone16", { app: "app_x" })
  .track(
    "app_x",
    (getOrder) => new XTrackBuilder(30, "phone", getOrder),
    (x) => {
      x.at("1s").post(
        { id: "alice", name: "Alice", handle: "alice" },
        "hello world",
      );
      x.at("3s").openNotifications();
    },
  );
```
