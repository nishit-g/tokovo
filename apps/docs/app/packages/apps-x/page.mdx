# @tokovo/apps-x

The X plugin package for Tokovo.

## Responsibilities

- X runtime reducer and initial state
- snapshot and initial-view schema ownership
- bootstrap validation and hydration
- feed, tweet detail, notifications, messages, and profile surfaces
- anchor integration
- authoring helpers and track builder support
- notification adapter and theme support

## Key Exports

- root package for authoring DSL and public X types
- `@tokovo/apps-x/plugin` for runtime registration
- `@tokovo/apps-x/contract` for bootstrap and event contracts
- `@tokovo/apps-x/runtime` for reducer state and selectors
- `@tokovo/apps-x/dsl` for track builders and authoring helpers
- `@tokovo/apps-x/lowering` for event lowering
- `@tokovo/apps-x/ui` for React surfaces
