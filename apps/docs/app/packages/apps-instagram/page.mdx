# @tokovo/apps-instagram

Instagram creator-core plugin for Tokovo.

## Responsibilities

- Instagram runtime reducer and selectors
- snapshot and initial-view schema ownership
- bootstrap validation and hydration
- home feed, story viewer, inbox, DM thread, notifications, profile, and composer surfaces
- semantic layouts, anchors, and theme support
- v2 DSL and notification integration

## Key Exports

- plugin registration: `registerInstagramPlugin`
- runtime state, selectors, and UI exports
- bootstrap types
- `InstagramTrackBuilder` and related DSL exports
