# @tokovo/apps-instagram

Instagram creator-core plugin for Tokovo.

## Responsibilities

- Instagram runtime reducer and selectors
- snapshot and initial-view schema ownership
- bootstrap validation and hydration
- home feed, story viewer, inbox, DM thread, notifications, profile, and composer surfaces
- semantic layouts, anchors, and theme support
- v2 DSL and notification integration

## Key Exports

- root package for authoring DSL and public Instagram types
- `@tokovo/apps-instagram/plugin` for runtime registration
- `@tokovo/apps-instagram/contract` for bootstrap, schema, and event contracts
- `@tokovo/apps-instagram/runtime` for reducer state and selectors
- `@tokovo/apps-instagram/dsl` for track builders and authoring helpers
- `@tokovo/apps-instagram/lowering` for event lowering
- `@tokovo/apps-instagram/ui` for React surfaces
