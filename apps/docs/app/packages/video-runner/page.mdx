# video-runner

Remotion render app for Tokovo.

`video-runner` renders directly from the checked-in episode registry.

## Responsibilities

- preview curated and legacy episode catalogs
- expose storybook-style browsing by app, system, stories, legacy, and tests
- run render-time plugin registration explicitly
- prepare and render episodes from checked-in code and assets

## Catalog Groups

The runner should surface:

- `Apps`
- `System`
- `Stories`
- `Legacy`
- `Tests`
