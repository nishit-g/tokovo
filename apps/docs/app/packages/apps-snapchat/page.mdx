# @tokovo/apps-snapchat

Snapchat plugin package for Tokovo.

## Responsibilities

- Snapchat runtime reducer
- snapshot and initial-view schema ownership
- bootstrap validation and hydration
- inbox, chat thread, and snap-viewer surfaces
- semantic layouts and anchors
- notification integration and DSL helpers

## Key Exports

- root package for authoring DSL and public Snapchat types
- `@tokovo/apps-snapchat/plugin` for runtime registration
- `@tokovo/apps-snapchat/contract` for bootstrap and event contracts
- `@tokovo/apps-snapchat/runtime` for reducer state and selectors
- `@tokovo/apps-snapchat/dsl` for track builders and authoring helpers
- `@tokovo/apps-snapchat/lowering` for event lowering
- `@tokovo/apps-snapchat/ui` for React surfaces
