# @tokovo/apps-linkedin

The LinkedIn plugin package.

## Responsibilities

- LinkedIn runtime reducer
- snapshot and initial-view schema ownership
- bootstrap validation and hydration
- LinkedIn React views
- feed and messaging surfaces
- profile, post detail, notification, and compose surfaces
- authoring helpers, themes, anchors, and notification integration

## Key Exports

- root package for authoring DSL and public LinkedIn types
- `@tokovo/apps-linkedin/plugin` for runtime registration
- `@tokovo/apps-linkedin/contract` for bootstrap and event contracts
- `@tokovo/apps-linkedin/runtime` for reducer state and selectors
- `@tokovo/apps-linkedin/dsl` for track builders and authoring helpers
- `@tokovo/apps-linkedin/lowering` for event lowering
- `@tokovo/apps-linkedin/ui` for React surfaces
