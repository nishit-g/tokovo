# @tokovo/apps-linkedin

The LinkedIn plugin package.

## Responsibilities

- LinkedIn runtime reducer
- snapshot and initial-view schema ownership
- bootstrap validation and hydration
- LinkedIn React views
- feed and messaging surfaces
- profile, post detail, notification, and compose surfaces
- authoring helpers, themes, anchors, and notification integration

## Key Exports

- plugin registration: `registerLinkedInPlugin`
- `LinkedInTrackBuilder`
- runtime state and selectors
- bootstrap types and UI exports
