# @tokovo/device-camera

`@tokovo/device-camera` provides Tokovo's camera layer.

## Responsibilities

- camera runtime behavior
- semantic-anchor framing support
- camera diagnostics and CLI tooling
- camera director integration with `@tokovo/compiler`

## Authoring Modes

- use `CameraDirectorPlugin` when a scene needs consistent automatic coverage
- use explicit camera DSL code when the scene needs hand-directed beats

## Tooling

The package exposes the `tokovo-camera` CLI for camera linting, preview, and migration support.

## Constraint

Tokovo currently treats camera as a single render-time system across the composition, even when multiple devices are visible.

## Key Exports

- runtime: `cameraReducer`
- plugin registration: `registerCameraPlugin`
- director layer: `CameraDirector`
- behavior presets and processors
- anchor and camera types
