# @tokovo/apps-typewriter

Typewriter simulation plugin for Tokovo.

## Responsibilities

- deterministic text runtime and lowering
- theme-driven paper, desk, and typography system
- anchor integration for text area and typing focus
- DSL track builder for document setup and typing actions

## Key Exports

- root package for authoring DSL and public theme helpers
- `@tokovo/apps-typewriter/plugin` for runtime registration
- `@tokovo/apps-typewriter/contract` for app identity and public track-event types
- `@tokovo/apps-typewriter/runtime` for reducer state
- `@tokovo/apps-typewriter/dsl` for track builders and authoring helpers
- `@tokovo/apps-typewriter/lowering` for event lowering
- `@tokovo/apps-typewriter/theme` for theme presets and theme helpers
- `@tokovo/apps-typewriter/ui` for React surfaces
