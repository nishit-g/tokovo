# @tokovo/apps-typewriter

Typewriter simulation plugin for Tokovo. Includes reducer/runtime, themed UI, deterministic text lowering, anchors, and DSL track builders.

## Install

```bash
pnpm add @tokovo/apps-typewriter
```

## Register

```ts
import { createTokovoRegistries, PluginManagerClass } from "@tokovo/react";
import { registerTypewriterPlugin } from "@tokovo/apps-typewriter";

const registries = createTokovoRegistries();
const pluginManager = new PluginManagerClass(registries.plugins);

registerTypewriterPlugin(pluginManager);
```

## Use In Episodes

```ts
import { episode } from "@tokovo/dsl";
import { TypewriterTrackBuilder } from "@tokovo/apps-typewriter";

episode("typewriter-demo", { fps: 30, duration: "20s" })
  .device("phone", "canvas", { app: "app_typewriter" })
  .track(
    "app_typewriter",
    (getOrder) => new TypewriterTrackBuilder(30, "phone", getOrder),
    (tw) => {
      tw.at("1s").initLetter({ subject: "Incident Report" });
      tw.span("2s", "8s").typeText("Everything looked calm until notifications exploded.");
      tw.at("9s").newline();
      tw.span("10s", "14s").typeText("Escalation acknowledged. Shipping patch now.");
    },
  );
```
