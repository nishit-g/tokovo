# @tokovo/apps-imessage

The iMessage plugin package.

## Responsibilities

- iMessage runtime reducer
- snapshot and initial-view schema ownership
- bootstrap validation and hydration
- iMessage React views
- layout, anchor, and notification integration
- authoring helpers and theme support

## Key Exports

- plugin surface from `plugin.ts`
- runtime and DSL exports
- bootstrap types
- UI exports, themes, and constants
