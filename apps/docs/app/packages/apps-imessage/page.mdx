# @tokovo/apps-imessage

The iMessage plugin package.

## Responsibilities

- iMessage runtime reducer
- snapshot and initial-view schema ownership
- bootstrap validation and hydration
- iMessage React views
- layout, anchor, and notification integration
- authoring helpers and theme support

## Key Exports

- root package for authoring DSL and public iMessage types
- `@tokovo/apps-imessage/plugin` for runtime registration
- `@tokovo/apps-imessage/contract` for bootstrap and event contracts
- `@tokovo/apps-imessage/runtime` for reducer state and selectors
- `@tokovo/apps-imessage/dsl` for track builders and authoring helpers
- `@tokovo/apps-imessage/lowering` for event lowering
- `@tokovo/apps-imessage/ui` for React surfaces
