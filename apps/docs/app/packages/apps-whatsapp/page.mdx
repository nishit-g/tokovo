# @tokovo/apps-whatsapp

The flagship messaging plugin package for Tokovo.

## Responsibilities

- WhatsApp runtime reducer
- snapshot and initial-view schema ownership
- bootstrap validation and hydration
- v2 lowering and DSL helpers
- chat, updates, calls, and channel surfaces
- semantic layout and anchor integration
- notification and audio behavior

## Key Exports

- root package for authoring DSL and public app types
- `@tokovo/apps-whatsapp/plugin` for runtime registration
- `@tokovo/apps-whatsapp/contract` for bootstrap and schema contracts
- `@tokovo/apps-whatsapp/runtime` for reducer state and selectors
- `@tokovo/apps-whatsapp/dsl` for track builders and authoring helpers
- `@tokovo/apps-whatsapp/lowering` for event lowering
- `@tokovo/apps-whatsapp/ui` for React surfaces
