# @tokovo/apps-whatsapp

The WhatsApp plugin package.

## Responsibilities

- WhatsApp runtime reducer
- WhatsApp snapshot and initial-view schema ownership
- snapshot/view validation and migration hooks
- WhatsApp React views
- chat and feed layout support
- anchor integration
- authoring helpers

## Key Exports

- plugin registration: `registerWhatsAppPlugin`
- reducer and selectors
- track builders: `WhatsAppTrackBuilder`, `WhatsAppPointBuilder`, `WhatsAppSpanBuilder`
- group helper: `GroupBuilder`
- layout helper: `computeChatLayout`
