# @tokovo/apps-whatsapp

**Priority: P1** | **Category:** Example App Plugin

WhatsApp messaging app plugin for Tokovo's phone simulation engine. Provides a high-fidelity WhatsApp chat interface with message sending/receiving, typing indicators, media sharing, and rich interaction capabilities.

For the complete anchor + DSL inventory used in production episode authoring, see [Anchors + DSL Reference](/guides/anchors-dsl-plugin-reference).

---

## Installation

```bash
npm install @tokovo/apps-whatsapp
```

## Quick Start

```typescript
import { episode } from "@tokovo/dsl";
import { WhatsAppTrackBuilder } from "@tokovo/apps-whatsapp";

episode("my-chat", { fps: 30, duration: "30s" })
  .device("phone", "iphone16", {
    app: "app_whatsapp",
    conversations: [
      { id: "dm_alice", name: "Alice", avatar: "/avatars/alice.png" },
    ],
  })
  .track(
    "app_whatsapp",
    (getOrder) => new WhatsAppTrackBuilder(30, "phone", "dm_alice", getOrder),
    (wa) => {
      // Receive message
      wa.at("1s").receive("Alice", "Hey! How are you?");

      // Show typing indicator
      wa.span("3s", "5s").typing("Alice");

      // Another message
      wa.at("5s").receive("Alice", "Want to grab coffee?");

      // Send message
      wa.span("7s", "9s").typing("me");
      wa.at("9s").send("Sure! When?");
    },
  )
  .build();
```

## Overview

The `@tokovo/apps-whatsapp` plugin is a **P1 (example app plugin)** that demonstrates Tokovo's plugin architecture. It provides a complete WhatsApp chat simulation including:

- **Realistic UI** - High-fidelity WhatsApp interface matching iOS design patterns
- **Message Types** - Text, images, videos, voice notes, GIFs, stickers, documents, contacts, locations
- **Interactions** - Typing indicators, read receipts, reactions, message forwarding
- **Group Chats** - Member management, admin controls
- **Navigation** - Chat screens, conversation list
- **Animations** - Smooth message transitions, typing indicators, scroll behavior

The plugin follows Tokovo's **three-tier plugin contract**:

1. **Tier A** - Core runtime (reducer, state, views)
2. **Tier B** - Lowering and layouts
3. **Tier C** - DSL extension (`wa.receive()`, `wa.send()`)

## API Reference

### Core Methods

#### `.receive(from, text, options?)`

Receive an incoming message from a contact.

```typescript
wa.at("1s").receive("Alice", "Hello!");

// With options
wa.at("1s").receive("Alice", "Reply to this", {
  messageType: "text",
  silent: false,
});
```

**Parameters:**

- `from: string` - Contact name
- `text: string` - Message content
- `options?: MessageOptions` - Optional configuration

#### `.send(text, options?)`

Send an outgoing message from "me".

```typescript
wa.at("3s").send("Hi there!");

// With reply
wa.at("5s").send("Got it!", {
  replyTo: messageId,
});
```

**Parameters:**

- `text: string` - Message content
- `options?: MessageOptions` - Optional configuration

#### `.typing(from, duration?)`

Show typing indicator for a contact, then hide it after duration.

```typescript
// Default 2s duration
wa.at("3s").typing("Alice");

// Custom duration
wa.at("5s").typing("Alice", "3s");
```

**Parameters:**

- `from: string` - Contact name
- `duration?: string` - How long to show typing (default: "2s")

#### `.react(messageId, emoji, from?)`

Add a reaction to a message.

```typescript
wa.at("10s").react("msg_123", "❤️", "Alice");
```

**Parameters:**

- `messageId: string` - Target message ID
- `emoji: string` - Reaction emoji
- `from?: string` - Who reacted (default: "me")

#### `.navigate(screen, conversationId?)`

Navigate to a screen within WhatsApp.

```typescript
wa.at("5s").navigate("chat", "dm_alice");
wa.at("10s").navigate("chats");
```

**Parameters:**

- `screen: 'chats' | 'chat' | 'story'` - Target screen
- `conversationId?: string` - Conversation to open (for 'chat' screen)

## Plugin Architecture

### Plugin Contract

The WhatsApp plugin implements Tokovo's full plugin contract:

```typescript
export const WhatsAppPlugin: TokovoPluginContract<"app_whatsapp"> = {
  // === TIER A: Identity ===
  id: "app_whatsapp",
  version: "2.0.0",
  displayName: "WhatsApp",

  // === TIER A: Runtime ===
  reducer: whatsappReducer,
  views: whatsappViews,
  createInitialState: createWhatsAppInitialState,

  // === TIER A: Event Routing ===
  eventKinds: [
    "MessageReceived",
    "MessageSent",
    "TypingStarted",
    "TypingEnded",
    // ... 30+ event types
  ],

  // === TIER B: Lowering ===
  v2Lowering: whatsappV2Lowering,

  // === TIER B: Layouts ===
  layouts: [
    {
      viewKind: "CHAT",
      computeLayout: computeChatLayout,
    },
  ],

  // === TIER C: DSL ===
  dsl: whatsappDsl,

  // === Assets ===
  assets: whatsappAssets,
  audioRules: whatsappAudioRules,
};
```

### Registration

```typescript
import { registerWhatsAppPlugin } from "@tokovo/apps-whatsapp";
import { createTokovoRegistries, PluginManagerClass } from "@tokovo/react";

const registries = createTokovoRegistries();
const pluginManager = new PluginManagerClass(registries.plugins);

// Register plugin with Tokovo
registerWhatsAppPlugin(pluginManager);
```

## Anchors

WhatsApp registers the following anchors for camera framing:

| Anchor          | Description         | Anchor Point | Use Case                  |
| --------------- | ------------------- | ------------ | ------------------------- |
| `message`       | Generic message     | (0.5, 0.5)   | Frame any message         |
| `message_me`    | Outgoing message    | (0.6, 0.5)   | Frame sent messages       |
| `message_other` | Incoming message    | (0.4, 0.5)   | Frame received messages   |
| `typing`        | Typing indicator    | (0.35, 0.5)  | Frame typing bubbles      |
| `input`         | Message input field | (0.5, 0.8)   | Frame text input          |
| `header`        | Chat header         | (0.5, 0.15)  | Frame contact name/avatar |
| `profile`       | Profile picture     | (0.2, 0.15)  | Frame avatar              |
| `device`        | Full device         | (0.5, 0.5)   | Frame entire phone        |

### Using Anchors

```typescript
episode("chat", { fps: 30, duration: "30s" }).camera((cam) => {
  // Frame the last message
  cam.at("5s").frameTo("message", {
    duration: "0.5s",
    easing: "easeInOut",
  });

  // Frame typing indicator
  cam.at("10s").frameTo("typing", {
    duration: "0.3s",
  });

  // Frame full device
  cam.at("15s").frameTo("device", {
    duration: "0.5s",
  });
});
```

## Complete Example

```typescript
import { defineEpisode } from "@tokovo/episodes";
import { episode } from "@tokovo/dsl";
import { WhatsAppTrackBuilder } from "@tokovo/apps-whatsapp";

export default defineEpisode({
  meta: {
    id: "coffee-chat",
    title: "Coffee Plans",
    category: "production",
  },
  config: {
    format: "1080x1920",
    durationInFrames: 900,
    apps: ["app_whatsapp"],
  },
  build: () =>
    episode("coffee-chat", {
      fps: 30,
      duration: "30s",
      title: "Coffee Plans",
    })
      .device("phone", "iphone16", {
        app: "app_whatsapp",
        conversations: [
          { id: "dm_sarah", name: "Sarah ☕", avatar: "/avatars/sarah.png" },
        ],
        os: {
          time: new Date("2024-12-20T09:00:00"),
          battery: 85,
          network: "5G",
        },
      })
      .track(
        "app_whatsapp",
        (getOrder) => new WhatsAppTrackBuilder(30, "phone", "dm_sarah", getOrder),
        (wa) => {
          // Opening message
          wa.at("1s").receive("Sarah", "Hey! Coffee later? ☕");

          // Show typing
          wa.span("3s", "5s").typing("me");

          // Reply
          wa.at("5s").send("Perfect! Where?");

          // Typing + response
          wa.span("7s", "9s").typing("Sarah");
          wa.at("9s").receive("Sarah", "The usual spot?");

          // Quick reply
          wa.span("10.5s", "11.5s").typing("me");
          wa.at("11.5s").send("Sounds good! 👍");

          // Send location
          wa.at("14s").receiveLocation("Sarah", {
            latitude: 37.7749,
            longitude: -122.4194,
            locationName: "Blue Bottle Coffee",
            locationAddress: "66 Mint St, San Francisco",
          });

          // React to location
          wa.at("16s").react("location_msg", "❤️", "me");

          // Final message
          wa.span("18s", "19s").typing("Sarah");
          wa.at("19s").receive("Sarah", "See you at 3pm!");
        },
      )
      .camera((cam) => {
        // Frame opening message
        cam.at("1s").frameTo("message", { duration: "0.5s" });

        // Frame typing indicator
        cam.at("7s").frameTo("typing", { duration: "0.3s" });

        // Frame location message
        cam.at("14s").frameTo("message", { duration: "0.5s" });

        // Zoom out to device
        cam.at("19s").frameTo("device", { duration: "0.5s" });
      })
      .build(),
});
```
