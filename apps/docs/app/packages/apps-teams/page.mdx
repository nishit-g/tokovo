# @tokovo/apps-teams

Microsoft Teams enterprise collaboration plugin for Tokovo.

## Responsibilities

- Teams runtime reducer and selectors
- snapshot and initial-view schema ownership
- bootstrap validation and hydration
- DMs, channels, threads, typing, notifications, and call surfaces
- semantic layouts, anchors, camera behaviors, and themes
- v2 DSL and lowering

## Key Exports

- plugin registration: `registerTeamsPlugin`
- runtime state, selectors, and layout helpers
- bootstrap types
- DSL helpers and Teams track builder
- theme/config/audio-rule exports
