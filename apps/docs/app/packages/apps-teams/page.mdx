# @tokovo/apps-teams

Microsoft Teams enterprise collaboration plugin for Tokovo.

## Responsibilities

- Teams runtime reducer and selectors
- snapshot and initial-view schema ownership
- bootstrap validation and hydration
- DMs, channels, threads, typing, notifications, and call surfaces
- semantic layouts, anchors, camera behaviors, and themes
- v2 DSL and lowering

## Key Exports

- root package for authoring DSL plus public config and theme helpers
- `@tokovo/apps-teams/plugin` for runtime registration
- `@tokovo/apps-teams/contract` for bootstrap, schema, and event contracts
- `@tokovo/apps-teams/runtime` for reducer state and selectors
- `@tokovo/apps-teams/dsl` for track builders and authoring helpers
- `@tokovo/apps-teams/lowering` for event lowering
- `@tokovo/apps-teams/ui` for React surfaces
- `@tokovo/apps-teams/theme` for theme tokens and providers
