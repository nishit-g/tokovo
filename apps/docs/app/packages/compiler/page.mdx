# @tokovo/compiler

The compilation layer between authored episode input and runtime replay.

## Responsibilities

- lower authored events and tracks into runtime data
- validate and migrate plugin-owned snapshots and initial views
- hydrate initial world state through plugin bootstrap contracts
- host compile-time plugins such as camera, keyboard, and typing-indicator helpers

`@tokovo/compiler` does not own app-domain bootstrap semantics. It orchestrates plugin-owned snapshot/view schemas; it does not synthesize chats, feeds, or threads on behalf of apps.

## Key Exports

- preparation: `prepareTrackEpisode`
- lowering helpers: `lowerTrackEvent`, `lowerTrackEvents`, `lowerEpisode`
- plugin contracts: `CompilerPlugin`, `CompilerContext`
- built-in plugins:
  - `CameraDirectorPlugin`
  - `AudioDirectorPlugin`
  - `OSDirectorPlugin`
  - `TypingIndicatorPlugin`
  - `KeyboardPlugin`
