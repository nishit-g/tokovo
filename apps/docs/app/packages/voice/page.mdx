# @tokovo/voice

Voice manifests, segment timing helpers, and voice-generation tooling.

## Responsibilities

- voice metadata exports
- timing-aware segment helpers
- sync and generation tooling
- audio-system integration

## Key Exports

- shared voice types
- DSL and render helpers
- generated script exports such as `drama_example`
- CLI commands exposed through the package scripts: `voice:generate` and `voice:sync`
